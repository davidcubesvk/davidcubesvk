Plugin in a nutshell:

This plugin is designed for the networks from the smallest, to the biggest ones. Offers and supports up to 27 lobbies. Includes non-flickering scoreboard, tablist, bossbar, announcer and a lot more! Lobby can be set as the whole world, or specified region, can be only for players with certain permission, can be closed, opened etc. This feature takes you away from having single server for each lobby! Not convinced yet? This plugin also has parkour system built-in, player hider, amazing inventory creating system and one big secret. Do you want to know it? I can't tell you. But, it is something like self-management, but if you download the plugin, you will find it more amazing as it can sound. Simply, it does all the work for you. Do you need to manage almost everything needed within the game chat? No problem. One big thing at the end, it is optimized for server even with 512MB RAM! Everything customizable. Are you using BungeeCord? This plugin has thought about this and introduced an extension for BungeeCord which allows players to connect to lobby from every server of your network! Continue reading this wiki and you will find more.

Why ExtremeLobby?

There are not many Lobby-System plugins, so I made a bomb filled with features and packed it within the one single plugin. This plugin has ultimate features that not every plugin has; just read the features below, but do not worry, it is not everything. You can configure the whole plugin, if there is something that is not configurable, contact me. All you need to do after downloading the plugin, configure it to fit your network perfectly and experience the features of this plugin!

Main features:

  • Designed for every network.
  • Supports from 1 to 27 lobbies.
  • Non-flickering scoreboard.
  • Tablist and bossBar support.
  • Announcer in chat, title or action bar.
  • Lobby can be world, specified region.
  • Lobby can be only for certain ranks (permission system).
  • Each lobby can have it's own weather and world time.
  • Rich lobby management system.
  • Amazing inventory creator, incredible action system.
  • Player hider, lobby selector...
  • Rich command management withing the game chat.
  • Parkour system.
  • PlaceholderAPI support.
  • Optimized for even 512MB RAM.
  • BungeeCord extension for connections from the whole network.
  • Multiple language support.
  • Configuration highly documented and sorted.
