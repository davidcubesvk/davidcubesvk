Base command (/extremeLobby or /el):

/el help
Description: displays the help page for command /el.
Permission: extremeLobby.command.help
/el reload
Description: reloads the plugin.
Permission: extremeLobby.command.reload
/el lobby
Description: sub-command for lobby management.
/el language
Description: sub-command for language management.
/el announcer
Description: sub-command for announcer management.
/el connector
Description: sub-command for connector management.
/el appearance
Description: sub-command for appearance management.
/el event
Description: sub-command for event management.
/el dynamic
Description: sub-command for dynamic management.
/el parkour
Description: sub-command for parkour management.

Lobby sub-command:

Parameters:

1.: <lobby> - lobby number to perform commands on

2.: commands - listed below

Commands:

/el lobby <lobby> spawn [world x y z yaw pitch]
Description: sets spawn to your location, or to location in command if [world x y z yaw pitch] arguments are specified. If sending from console, these arguments are required.
Permission: extremeLobby.command.lobby.spawn
/el lobby <lobby> region <1|2> [x y z]
Description: sets region bound 1/2 to your location, or to location in command if [x y z] arguments are specified. If sending from console, these arguments are required.
Permission: extremeLobby.command.lobby.region
/el lobby <lobby> open
Description: opens lobby.
Permission: extremeLobby.command.lobby.open
/el lobby <lobby> close
Description: closes lobby.
Permission: extremeLobby.command.lobby.close
/el lobby <lobby> restart
Description: closes and opens lobby.
Permission: extremeLobby.command.lobby.restart
/el lobby <lobby> premium <true|false>
Description: sets lobby premium mode.
Permission: extremeLobby.command.lobby.premium
/el lobby <lobby> permission <permission>
Description: sets permission domain for premium lobby (extremeLobby.premium.<permission>).
Permission: extremeLobby.command.lobby.permission
/el lobby <lobby> type <WORLD|REGION>
Description: sets lobby type.
Permission: extremeLobby.command.lobby.type
/el lobby <lobby> slots <slots>
Description: sets lobby slots.
Permission: extremeLobby.command.lobby.slots
/el lobby <lobby> reserved <slots>
Description: sets reserved slots.
Permission: extremeLobby.command.lobby.reserved
/el lobby <lobby> void [y]
Description: sets void position to your Y-location, or to Y-location in command if [y] argument is specified. If sending from console, this argument is required.
Permission: extremeLobby.command.lobby.void
/el lobby <lobby> info
Description: shows info.
Permission: extremeLobby.command.lobby.info
/el lobby <lobby> reload
Description: reloads cached lobby configuration. Can be used only if lobby is closed.
Permission: extremeLobby.command.lobby.reload
/el lobby <lobby> help
Description: shows the help page for this sub-command.
Permission: extremeLobby.command.lobby.help

Language sub-command:

Parameters:

1.: commands - listed below

Commands:

/el language list <page>
Description: lists all available languages.
Permission: extremeLobby.command.language.list
/el language default
Description: displays default language.
Permission: extremeLobby.command.language.default
/el language get <player>
Description: gets player's language.
Permission: extremeLobby.command.language.get
/el language set <player> <language>
Description: sets player's language.
Permission: extremeLobby.command.language.set
/el language help
Description: shows the help page for this sub-command.
Permission: extremeLobby.command.language.help

Announcer sub-command:

Parameters:

1.: commands - listed below

Commands:

/el announcer type <PUBLIC|LOBBY|SINGLE>
Description: sets announcer type.
Permission: extremeLobby.command.announcer.type
/el announcer enable <true|false>
Description: enables or disables announcer.
Permission: extremeLobby.command.announcer.enable
/el announcer reload
Description: reloads announcer.
Permission: extremeLobby.command.announcer.reload
/el announcer <chat|actionBar|title> frequency <frequency>
Description: sets announcing frequency for sub-announcers.
Permission: extremeLobby.command.announcer.sub.frequency
/el announcer <chat|actionBar|title> enable <true|false>
Description: enables or disables sub-announcers.
Permission: extremeLobby.command.announcer.sub.enable
/el announcer help
Description: shows the help page for this sub-command.
Permission: extremeLobby.command.announcer.help

Connector sub-command:

Parameters:

1.: commands - listed below

Commands:

/el connector <lobby> <player>
Description: connects player to specific lobby number (as /lobbyX).
Permission: extremeLobby.command.connector.specific
/el connector auto <player>
Description: connects player to random lobby (as /lobby).
Permission: extremeLobby.command.connector.auto
/el connector help
Description: shows the help page for this sub-command.
Permission: extremeLobby.command.connector.help
