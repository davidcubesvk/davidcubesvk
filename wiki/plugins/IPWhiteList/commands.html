The base command is /ipWhiteList (or shortened: /ipwl). All commands can be ran by player and console. Permission ipWhiteList.* is an universal permission for all commands.

Command: Permission: Arguments: Description:
/ipwl add <ip> ipWhiteList.add <ip> - IP to add Adds the specified IP to the whitelist.
/ipwl remove <ip> ipWhiteList.remove <ip> - IP to remove Removes the specified IP from the whitelist.
/ipwl list [page] ipWhiteList.list [page] - page to display Lists the whitelisted IPs on the specified page (or 1st page if not specified).
/ipwl setup ipWhiteList.setup Toggles the setup mode.
/ipwl reload ipWhiteList.reload Reloads the plugin, excluding connection with the extension.
/ipwl help ipWhiteList.help Displays help page.

<> - required argument

[] - optional argument
