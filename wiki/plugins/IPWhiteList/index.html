Are you tired of hackers griefing your servers or do you want to prevent this? Well, you arrived on the right spot. IPWhiteList in combination with it's backend server plugin IPWhiteListBukkit is yet unbypassed plugin trusted by many big networks (you can see the list on plugin page). Tested by several users with different bypass-clients, still unbreakable shield for your network. User-friendly and easy to set up, supports multi-proxy servers.

Why IPWhiteList?

IPWhiteList is the best choice for keeping bad people away from your server currently available on the internet. However, it's principle is really simple - restrict IPs with which player can connect to your server to only several IPs specified by you.

Key features:

  • Whitelist IPs, ports and hostnames.
  • Commands included.
  • Setup mode.
  • Plugin API for developers.
  • Can be used also on servers with dynamic IP address.
  • Update checker available.
  • Support for multi-proxy servers.
