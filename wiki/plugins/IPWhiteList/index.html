Plugin in a nutshell:

This plugin is designed to keep your server safe. Includes many commands for live administration, but also socket connections to it's extrension IPWhiteListBukkit. With this extension you are granted with 100% resistance against bypassers. You are allowed to whitelist IPs, but also hostnames and ports.

Why IPWhiteList?

Lightweight, fast and secures your server. Do you need something more?

Main features:

  • Whitelist IPs, ports and hostnames.
  • Commands for real-time administration.
  • Setup mode.
  • Plugin API for developers.
  • Can be used also on servers with dynamic IP address.
  • Update checker available.
  • Full resistance with extension.
