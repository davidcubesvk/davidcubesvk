Plugin in a nutshell:

This plugin is designed to keep your server safe. Includes many commands for live administration, but also socket connections to it's extrension IPWhiteListBukkit. If using this extension plugin, you are granted with resistance against bypassers; cheat clients always get better and better, if you find new method, make sure to tell me! You are allowed to whitelist IPs, but also hostnames and ports.

Why IPWhiteList?

IPWhiteList is the best choice for keeping bad people away from your server. The main purpose of this plugin is to restrict the IPs that players can connect with (e.g. your proxy). It is required to have IPWhiteListBukkit installed, so you don't have to worry about your server anytime.

Main features:

  • Whitelist IPs, ports and hostnames.
  • Commands for real-time administration.
  • Setup mode.
  • Plugin API for developers.
  • Can be used also on servers with dynamic IP address.
  • Update checker available.
  • Full resistance with extension.
