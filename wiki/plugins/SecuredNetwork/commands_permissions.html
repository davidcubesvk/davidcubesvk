Plugin commands:

The base command is /securedNetwork, or /sn. Parent permission securedNetwork.* is also available. The plugin contains only one command, so nobody can disable the protection in-game:

Command: Permission: Description:
/sn securedNetwork.command Reloads the plugin (except the connection type).

Permissions:

Parent permission securedNetwork.* is also available.

Permission: Description:
securedNetwork.updater Everytime you join the server, you will be notified about the latest plugin version.
