Please note that the API is available only if using SecuredNetwork's API on the proxy.

How to use:

  • Add SecuredNetwork-0.0.jar into project sources.
  • Add depend or softdepend for the SecuredNetwork plugin into your plugin's plugin.yml:

    Adding depend or softdepend to your plugin.yml

    depend: [SecuredNetwork]
    OR
    softdepend: [SecuredNetwork]
  • Get the API class:

    Getting the API class

    SecuredNetworkAPI.getInstance();
