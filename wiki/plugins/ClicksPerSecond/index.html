ClicksPerSecond plugin brings a lot of fun to your server. Players are able to measure their CPS and then compete with each-other. Scoreboards and player statistics are also implemented. Do you want to limit the CPS? Absolutely no problem. And a lot more available after downloading the plugin!

Why ClicksPerSecond?

ClicksPerSecond is here to replace mods and datapacks with stilish and modern way of measuring CPS. The plugin itself has high performance and optimized code, supporting versions 1.7 - 1.16. Implementing innovative action-based system, it doesn't contain any messy configuration and you can do pretty much everything you want to.

Key features:

  • Innovative action-based system.
  • Database support.
  • Custom commands.
  • Plugin API for developers.
  • Scorebaords, player statistics.
  • CPS limiter.
  • Allowed worlds (world limiter).
  • 100% customizable.
