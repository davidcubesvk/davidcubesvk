How to use:

  • Add ClicksPerSecond-0.0.jar into project sources.
  • Add depend or softdepend for the ClicksPerSecond plugin into your plugin's plugin.yml:

    Adding depend or softdepend to your plugin.yml

    depend: [ClicksPerSecond]
    OR
    softdepend: [ClicksPerSecond]
  • Get the API class:

    Getting the API class

    ClicksPerSecondAPI.getInstance();
