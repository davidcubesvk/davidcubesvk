Commands:

The base command is /ipWhiteListBukkit and /ipwlb . All commands can be ran by player and console. you can also use permission ipWhiteListBukkit.* which will grant you permission for all commands.

Command: Permission: Arguments: Description:
Base command ipWhiteListBukkit.command Reloads the connection to IPWhiteList. Not config!
