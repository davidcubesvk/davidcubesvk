The base command is /ipWhiteListBukkit (or shortened: /ipwlb). All commands can be ran by player and console. Permission ipWhiteListBukkit.* is an universal permission for all commands.

Command: Permission: Arguments: Description:
Base command ipWhiteListBukkit.command Reloads the connection with IPWhiteList. Does not reload the connection type!
