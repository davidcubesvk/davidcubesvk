Plugin commands:

The base command is /repair. Permission repairItem.* is an universal permissions for all commands. Here's a quick summary (please note that actual commands can be reconfigured, commands in the table are shown by default configuration) :

Command: Permission: Description:
/repair all [target] repair.all / repair.target.all Repairs all items in your/target's inventory.
/repair inventory [target] repair.inventory / repair.target.inventory Repairs inventory items (except armor contents, including both hands) in your/target's inventory.
/repair armor [target] repair.armor / repair.target.armor Repairs armor contents in your/target's inventory.
/repair hotbar [target] repair.hotbar / repair.target.hotbar Repairs hot-bar items (including both hands) in your/target's inventory.
/repair bothhands [target] repair.bothhands / repair.target.bothhands Repairs items in your/target's both hands (off-hand only if supported - server version 1.9 and newer).
/repair mainhand [target] repair.mainhand / repair.target.mainhand Repairs items in your/target's main-hand.
/repair offhand [target] repair.offhand / repair.target.offhand Repairs items in your/target's off-hand (if supported - server version 1.9 and newer)

[] - optional argument
